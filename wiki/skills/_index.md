# Skills Hub

Skills catalogs the six upstream frontend design skills that Gogh stacks into one advisory system.

## Catalog

### gogh/taste-skill

- [[Taste Skill (Project)]] - Taste Skill is the Gogh stack anchor for macro frontend taste prompting: use it for landing pages, portfolios, and redesigns that need brief-first direction.

### gogh/mifb

- [[Make Interfaces Feel Better (Skill)]] - Make Interfaces Feel Better is the MIFB micro-interaction execution skill for turning small interface details into visible polish.

### gogh/impeccable

- [[Impeccable (Toolchain)]] - Impeccable is the Gogh stack's toolchain enforcement layer for frontend design agents.

### gogh/frontend-design

- [[Anthropic Frontend Design Skill]] - Anthropic Frontend Design Skill is the official frontend-design baseline skill for Claude-style agent workflows, and its strongest evidence-backed use is early aesthetic direction before stricter rulebooks, detectors,...

### gogh/ui-ux-pro-max

- [[UI UX Pro Max (Skill)]] - UI UX Pro Max is the retrieval database skill in the Gogh stack, using searchable CSV data and BM25 ranking for style, palette, typography, UX, chart, and stack guidance.

### gogh/web-design-guidelines

- [[Vercel Web Design Guidelines]] - Vercel Web Design Guidelines is Gogh's runtime-fetched audit skill for checking existing UI code against Vercel's Web Interface Guidelines.

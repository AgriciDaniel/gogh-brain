# Concepts Hub

Concepts explains the design theory, mechanisms, and reusable models behind the Gogh stack.

## Catalog

### gogh/taste-skill

- [[Anti-Laziness Research]] - Anti-Laziness Research is the Taste Skill repository evidence folder behind output-skill completion discipline, not direct proof of visual taste quality.
- [[DESIGN_VARIANCE]] - DESIGN_VARIANCE is the Taste Skill dial that controls how far a layout moves from predictable symmetry toward asymmetric composition.
- [[Image-First Pipeline]] - Image-First Pipeline is Taste Skill's workflow of generating or collecting visual references, analyzing them, and implementing code against a concrete target.
- [[MOTION_INTENSITY]] - MOTION_INTENSITY is the Taste Skill dial that decides whether a page stays static, uses fluid transitions, or carries advanced choreography.
- [[The Three Dials]] - The Three Dials are Taste Skill v2 scalar controls for layout variance, motion intensity, and visual density, with a baseline of 8, 6, and 4 as of 2026-07-07.
- [[VISUAL_DENSITY]] - VISUAL_DENSITY is the Taste Skill dial that sets how much information and spacing a viewport should carry.

### gogh/mifb

- [[Concentric Radii]] - Concentric Radii is the MIFB surface rule that nested rounded shapes should keep mathematically related curves so the interface does not feel subtly off.
- [[Interruptible Animation]] - Interruptible Animation is the MIFB rule that interactive state changes should retarget smoothly instead of restarting from a fixed keyframe timeline.
- [[Optical Alignment]] - Optical Alignment is the MIFB rule that visual balance can override geometric centering when icons, triangles, and asymmetric shapes look off.
- [[Press Feedback and Hit Areas]] - Press Feedback and Hit Areas is the MIFB interaction rule pair: buttons should feel tactile at scale 0.96, and controls need usable hit areas.

### gogh/impeccable

- [[Deterministic Design Detection]] - Deterministic design detection is Impeccable's no LLM enforcement path for repeatable frontend anti-pattern checks.
- [[Impeccable Design Contracts]] - Design contracts are Impeccable's persistent project context files for keeping agent design work aligned across commands.

### gogh/ui-ux-pro-max

- [[Retrieval Database Skills]] - Retrieval Database Skills are agent skills that ship searchable data and retrieval scripts so the agent can pull a relevant design slice on demand instead of carrying every rule in context.

### gogh/web-design-guidelines

- [[Runtime Rule Fetch]] - Runtime Rule Fetch is the pattern where a skill retrieves its operative rules at review time instead of packaging those rules inside the skill wrapper.

### gogh/stack

- [[Agent Skills Format]] - Agent Skills Format is the portable SKILL.md folder pattern that lets Gogh treat design guidance as loadable operating context.
- [[Design Skills Mechanism Map]] - Design Skills Mechanism Map is the stack anchor: Gogh treats the six skills as four mechanisms, not as one interchangeable prompt pile.

### gogh/theory

- [[AI Slop]] - AI slop is the default cluster of same-looking frontend choices produced when a model samples from common training-data patterns instead of a project-specific design read.
- [[Constraint Beats Coaxing]] - Constraint Beats Coaxing is the stack thesis that explicit constraints outperform generic requests to make a frontend beautiful.
- [[Design Review as Infrastructure]] - Design Review as Infrastructure means moving frontend quality checks into reusable agent instructions, audits, and detector layers before the agent claims completion.
- [[Distributional Convergence]] - Distributional Convergence is the theory that unguided models produce same-looking frontends because they sample from common training-data patterns rather than from a project-specific design thesis.
- [[Encoded Design Principles]] - Encoded Design Principles explains how familiar design craft becomes agent-readable constraints, checks, and thresholds.
- [[Taste as the Moat]] - Taste as the Moat is the contested thesis that judgment matters more as generation gets cheaper, while dissent says taste can become a threshold rather than a permanent moat.
